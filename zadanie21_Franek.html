Co znaczy skrót ID? ID to skrót od „identifier” – oznacza identyfikator, który jednoznacznie identyfikuje dany element w kodzie HTML/CSS.
Jaki jest cel stosowania ID? ID pozwala wyróżnić jeden, konkretny element na stronie i nadać mu unikalny styl lub działanie.
Jakie jest znaczenie znaku # Znak # służy do definiowania identyfikatora w arkuszu CSS.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID następuje poprzez użycie atrybutu `id="nazwa"` w znaczniku HTML oraz zdefiniowanie stylu CSS jako `#nazwa { ... }`.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia. • ID powinno być unikatowe w całym dokumencie.
• Wielokrotne użycie tego samego ID to błąd, ale nie powoduje błędów działania strony.
• Jeśli nie mamy pewności, czy element wystąpi więcej niż raz, lepiej użyć klasy.
